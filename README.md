# Cubic Turnkey Website

Basic React site scaffold for the Cubic Turnkey brand, now structured around the sitemap in `sitemap.md`.

## Run locally

1. Install dependencies with `npm install`
2. Start the dev server with `npm run dev`
3. Build for production with `npm run build`

## Notes

- Brand colors and font direction were taken from `style.md`
- The header uses the uploaded logo-with-text lockup and the site favicon is served from `public/favicon.png`
- The current app is a multi-page skeleton with routes for Home, About, Services, Quality & Compliance, Careers, Gallery, and Contact
